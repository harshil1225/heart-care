# ❤️ Heart Care 

## 📖 Description
The Heart Care System is a web-based application designed to help manage heart-related healthcare services. It allows patients to book appointments, consult doctors, monitor health records, and receive proper guidance for heart care.

This system is useful for patients, doctors, and administrators to manage medical services efficiently and ensure better healthcare support.

---

## 🎯 Features

### 👤 User Management
- Patient Registration and Login  
- Doctor Login  
- Update Profile  
- Change Password  
- Forgot Password  

### 🏥 Doctor Management
- Add and manage doctors  
- View doctor details  
- Assign patients  

### 📅 Appointment Management
- Book appointment  
- View appointment schedule  
- Cancel appointment  

### ❤️ Health Monitoring
- Store patient medical records  
- Track heart-related health data  
- View reports  

### 💊 Treatment & Guidance
- Provide prescriptions  
- Suggest treatments  
- Health tips for patients  

### 💳 Payment System
- Online Payment (UPI / Net Banking)  
- Offline Payment (Cash)  

### ⭐ Feedback System
- Add feedback  
- View feedback  
- Update/Delete feedback
